# Módulo: Sistema Integral de Mantenimiento Automotriz (SIMA)

## Descripción de la aportación
Este módulo contiene la lógica del backend para la gestión de alertas e inventario del sistema SIMA.

## Funcionalidades implementadas
* **Alertas del Sistema**: Lógica para la verificación de stock crítico y generación de respuestas JSON.
* **Tabla de Inventarios (Backend)**: Estructura base para la consulta de estados de inventario.
* **Panel de Administración**: Configuración inicial de las rutas y controladores para la gestión del panel central.

## Tecnologías utilizadas
* Framework: Laravel
* Lenguaje: PHP
